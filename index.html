Sorting Algorithm

Bubble Sort

Bubble sort is a simple sorting algorithm that repeatedly compares adjacent elements in a list, until the entire list is sorted.

Insertion Sort

An insertion sort sorts an array by successively inserting the elements of the array into the sorted portion.

Selection sort Sort

Selection sort is a simple algorithm that sorts a list by repeatedly finding the smallest or largest element in the unsorted part of the list and moving it to the sorted part.

Quick Sort

Quick sort is a sorting algorithm that breaks an array into smaller arrays to sort it.

Merge Sort

Merge sort is a sorting algorithm that uses the "divide and conquer" strategy to sort a list of elements by repeatedly dividing the list into smaller sublists until each sublist contains only one element, then merging those sublists back together in a sorted manner, resulting in a completely sorted list.

Heap Sort

Heap sort is a sorting algorithm that utilizes a data structure called a "heap" to efficiently organize and sort elements in an array by repeatedly removing the largest element from the heap (which is always at the root) and placing it at the end of the sorted array.

Linked List

Singly Linked List

Singly linked list is a linear data structure in which the elements are not stored in contiguous memory locations and each element is connected only to its next element using a pointer.

Doubly Linked List

A doubly linked list is a linked data structure that consists of a set of sequentially linked records called nodes. Each node contains three fields: two link fields and one data field.

Circular Linked List

A circular linked list is a type of linked list data structure where the last node connects back to the first node, forming a circular loop.

Stack

Stack

A Stack is a linear data structure that holds a linear, ordered sequence of elements. A Stack works on the LIFO process (Last In First Out), i.e., the element that was inserted last will be removed first.

Queue

Queue

The Queue is a linear data structure or an abstract data type. Queue follows the FIFO - “first in, first out” method to process the data.

Dequeue

A deque, also known as a double-ended queue, is an ordered collection of items similar to the queue. It has two ends, a front and a rear.

Circular queue

A circular queue is a linear data structure that connects the last element to the first, forming a circle. This structure is also known as a ring buffer.
